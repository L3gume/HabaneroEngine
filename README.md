# VapeEngine

A simple C++ 3D game engine for Linux (and maybe windows) made with OpenGL

## Team

* Justin Tremblay
* Evan Laflamme

## Dependencies

* C++11 (May change to C++14 or even C++17)
* glfw-x11 3.2.1-1 (Available on Arch repos)
* glew 2.1.0-1 (Available on Arch repos)
* cmake 3.9
* glm 0.9.8.5-1
